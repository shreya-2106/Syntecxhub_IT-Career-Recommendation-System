## IT-Career-Recommendation-System
A Python-based rule-based expert system for IT career guidance. Users provide their skills and interests, and the system recommends suitable IT careers using forward chaining and multi-step reasoning. Includes conflict resolution and explanation of the inference steps.

## Features
- Personalized career suggestions
- Skill & interest-based analysis
- Covers multiple IT domains
- Simple and user-friendly interface
- Rule-based decision system

## How It Works
The system takes user input through a set of questions related to skills and interests. Based on predefined rules, it evaluates responses and suggests the most suitable IT career path.

##  Use Cases
- Students exploring IT careers
- Beginners in tech

## Technologies Used
- Python

## How to Run
- Download the file
- Open terminal
- Run: python main.py
